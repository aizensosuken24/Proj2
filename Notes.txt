# NammaRoute v3.0 — Upgrade Notes

## Files Changed
| File | Status | Notes |
|------|--------|-------|
| `db.js` | **NEW** | All data: stations, fares, graph, routing functions |
| `script.js` | **Upgraded** | Uses db.js; Dijkstra routing; accurate fares |
| `index.html` | **Upgraded** | Loads db.js before script.js; expanded station list |
| `style.css` | Unchanged | No changes needed |

## Deployment Steps (GitHub Pages)
1. Add `db.js` to your repo root (same folder as `index.html`)
2. Replace `script.js` with the new version
3. Replace `index.html` with the new version
4. Commit & push — GitHub Pages auto-deploys

```bash
git add db.js script.js index.html
git commit -m "v3.0: accurate Dijkstra routing + db.js data layer"
git push
```

## What's New in v3.0

### db.js — New Data Layer
- **57 metro stations** across all 3 lines with accurate lat/lng
- **Real station-to-station distances** along track (from HMRL data)
- **Dijkstra shortest-path** algorithm for accurate metro route calculation
- **Official HMRL Metro fare slabs** (₹10–₹60, 11 distance bands)
- **Official TSRTC fare slabs** — Ordinary (₹6–₹25) and Express/Pushpak (₹8–₹30)
- **Smart Card discount**: 10% off metro, minimum ₹10
- **Crowd profiles** per mode and time period
- **CO₂ savings** vs auto-rickshaw (105g/km auto vs 18g/km metro)
- **8 major TSRTC bus routes** with stops mapped

### script.js — Routing Engine Upgrade
- Routes now use **Dijkstra on the real metro graph** instead of haversine guesses
- Metro fare calculated from **actual track distance**, not straight-line distance
- **Line-aware steps**: Red Line 🔴 / Blue Line 🔵 / Green Line 🟢 labelled per segment
- **Interchange detection**: auto-detects line changes at Ameerpet
- **Access legs**: if source/dest isn't on metro, calculates walk/auto to nearest station
- Crowd bar now shows **percentage fill** based on time-of-day profile
- Map now uses **CartoDB dark tiles** (matches app aesthetic better)
- URL sharing with **?r= param** for restore-on-load

### index.html — Station List
- Expanded from ~22 to **57 stations** across all 3 lines
- Optgroups labelled with line colour emoji (🔴🔵🟢)
- Ameerpet marked as interchange (⇄)
- Footer updated to v3.0 with "Dijkstra Metro Routing" note
